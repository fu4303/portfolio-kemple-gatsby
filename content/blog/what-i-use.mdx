## Software

- [1password](https://1password.com/) for managing my passwords, can share vaults to manage multiple users of one account
- [Adobe Creative Cloud](https://www.adobe.com/creativecloud.html) for most A/V and digital graphics needs
  - **Light Room** for photo cropping, rotating, color grading, can purchase or find presets to create stylistic photography quickly
  - **Photoshop** for photo editing, removing backgrounds or people, touch ups, creating digital ads and marketing materials like flyers, banners, etc
  - **After Effects** for adding visual effects to video
  - **Premiere Pro** for editing quality videos for vlogging, screencasting, streaming, etc
  - **Audition** for editing movie audio and podcasts
  - **Media Queue** for managing exports of videos and audio from Adobe apps
- [Alfred](https://www.alfredapp.com/) for quick search on computer, web, and for workflows (automate tasks)
  - [Div](http://www.packal.org/workflow/div) is an Alfred plugin that lets you resize windows to specific dimensions, I use it to size windows to 1920x1080 when I stream
- [Discord](https://discordapp.com/) for communication, mostly play and a bit of career focus chat
- [Dropbox](https://dropbox.com) for cloud based file storage
- [Folx](https://mac.eltima.com/download-manager.html) for managing torrents
- [Giphy Capture](https://giphy.com/apps/giphycapture) for making sweet sweet gifs
- [iTerm2](https://www.iterm2.com/) for a better terminal experience
- [krisp.ai](https://krisp.ai) for removing background noise from my audio during screencasting, streaming, or video conferencing
- [Magnet](https://magnet.crowdcafe.com/) for organizing my windows
- [Quip](https://quip.com/) for documents and spreadsheets
- [Screenflow](https://www.telestream.net/screenflow/) for recording screencasts, can also be used for minor video editing instead of using Adobe Premiere Pro
- [Sketch](https://www.sketch.com/) for digital graphics needs (I prefer Adobe but a lot of folks use Sketch so good to have/know)
- [Slack](https://slack.com) for communication, mostly work and career focused chat
- [Stream Deck](https://www.elgato.com/en/gaming/downloads) for using the Stream Deck mobile app, it let's you control your OBS streams easily
- [Streamlabs OBS](https://streamlabs.com/) for streaming
- [VLC](https://www.videolan.org/vlc/index.html) for watching videos
- [VSCode](https://code.visualstudio.com/) for coding
- [Zoom](https://zoom.us/) for video calls, workshops, online talks or conferences

## VSCode Setup

![VSCode screenshot](../assets/vscode-screenshot.png)

### Theming

- **Icons** are from Material Icon Theme
- **Syntax highlighting** is from Mirage theme in Dainty extension
- **Font** is MonoLisa
- **Terminal** is spaceship theme for ZSH

### Extensions

- **Apollo GraphQL** for intellisense for GraphQL and request timing estimates
- **Bracket Pair Colorizer 2** to track bracket closings
- **Dainty - themes** for the steazy themes, I use Mirage
- **GitHub Pull Requests** to manage PRs from within VSCode
- **GitLens** for indepth git information including inline commit info
- **Import Cost** see how big imports used in your projects are
- **indent-rainbow** to track indentation
- **Live Share** to share my editor with other devs for pairing
- **Material Icon Theme** to stylize the overall theme of VSCode and get icons for file types
- **MDX** for syntax highlighting
- **MDX Preview** to preview MDX docs including embedded components
- **Path Intellisense** get intellisense on path based imports, great for larger projects
- **Peacock** adds a color theme to VSCode windows so you can tell them apart when you have multiple open
- **Prettier** format code!
- **Streamdeck for VSCode** to use my streamdeck to control VSCode
- **vscode-spotify** control spotify from VSCode
